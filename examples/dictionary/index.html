Scopes can also be layered in a hierarchy, so that child scopes have access to parent scopes, but not vice versa. Source: https://developer.mozilla.org/en-US/docs/Glossary/Scope

TypeScript (TS)

TypeScript is a superset of [JavaScript](#javascript-js) that is compiled into plain JavaScript that can be interpreted by any browser. TypeScript its self cannot be interpreted by browsers, so it is a kind of dialect of JavaScript. A typical new feature is to *type-set* variables: ```ts const foo: string[] // type-setting `foo` to an array of strings const foo = ['foo', 'bar', 'baz'] // setting value of `foo` ``` *Source: https://www.typescriptlang.org/index.html*
